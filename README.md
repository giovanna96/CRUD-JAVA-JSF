## Lista de Compras 
* Foi desenvolvido uma aplicação para simular uma lista de compras com as opções de adicionar, editar, remover e visualizar itens(produtos).
* Dentro dos itens do desafio foram utilizador jsf,Hibernate e JPA e bootstrap.
* Para rodar a aplicação foi utilizado o servidor Tomcat versão 9.0 e banco de dados Postgres
* Ao clonar a aplicação é necessário alterar as informações de acesso ao banco dados no arquivo persistence.xml
